## HowToRun

1. install requirements
    ```bash
    pip install -r requirements.txt
    ```

2. run code
    ```bash
    cd value-based/tabular

    # test sarsa code
    python -m sarsa
    # if running in terminal without display, use blow command
    xvfb-run -a -s "-screen 0 1400x900x24" python -m raw_dqn
    ```
